## Quick Start

### For Visitors
Simply visit [https://men0456.github.io](https://men0456.github.io) to explore the portfolio.

### For Developers/Contributors
1. Clone this repository
2. Launch in a dev container or ensure you have Quarto installed
3. Run `quarto preview` to see local changes
4. Make edits and push to deploy automatically

## What's Included

- **Devcontainer** with Python, Quarto, and GitHub Copilot pre-configured
- **GitHub Actions** workflow for automatic deployment to GitHub Pages
- **Python** environment with common data science libraries
- **Quarto** for publishing interactive documents and websites
